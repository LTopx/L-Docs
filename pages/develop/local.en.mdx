# Local Development

## Node Environment Requirements

NodeJS >= 18

## Development Process

**1. Install PNPM**

If you haven't installed or used `pnpm` before, you can install it by running the following command.

```bash
npm install pnpm -g
```

**2. Install Dependencies**

```bash
pnpm i
```

**3. Configure Environment Variables**

Rename .evn.local.demo to .env.local

**4. Run the Project**

```bash
pnpm dev
```

**5. Build the Project**

```bash
pnpm build && pnpm start
```
