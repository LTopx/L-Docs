# 本地开发

## Node 环境要求

NodeJS >= 18

## 开发流程

**1. 安装 PNPM**

如果你之前没有安装或使用过`pnpm`，你可以通过运行以下命令来安装它。

```bash copy
npm install pnpm -g
```

**2. 安装依赖**

```bash copy
pnpm i
```

**3. 配置环境变量**

将 .evn.local.demo 重命名为 .env.local

**4. 运行项目**

```bash copy
pnpm dev
```

**5. 打包项目**

```bash copy
pnpm build && pnpm start
```
