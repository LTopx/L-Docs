# Le-AI API

## 什么是 Le-AI API

为了方便那些没有 OpenAI 账户或者由于网络原因以及其他原因不方便使用 OpenAI API 的用户，我们提供了 Le-AI API。这是一个基于 OpenAI API 的封装，可以直接使用，无需 OpenAI 账户。

你除了能在 Le-AI 中使用之外，你还可以在其他类 ChatGPT 软件使用。例如：

- ChatGPT Next Web
- ChatBox
- OpenCat
- AMA

访问链接：<Link href={env.NEXT_PUBLIC_APIPROXY_LINK} className="text-[#006be6] dark:text-[#008ae6] underline" target="_blank">{env.NEXT_PUBLIC_APIPROXY_LINK}</Link>

## 功能特性

### 支持 Chat/Images/Audio API

目前支持 OpenAI API 的三个接口，分别是 **[/chat](https://platform.openai.com/docs/api-reference/chat)**, **[/images](https://platform.openai.com/docs/api-reference/images)**, **[/audio](https://platform.openai.com/docs/api-reference/audio)**。

API 调用方式与官方完全一致。

### 快速查看 API 消耗

Le-AI API 会记录每次调用时消耗 tokens 的情况，并且将其展示在首页中方便查看。包括月度消耗以及每次调用时的消耗记录。
