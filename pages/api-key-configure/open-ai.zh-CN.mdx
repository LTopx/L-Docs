# OpenAI API Key 配置

您可以在未登录的情况下使用 Le-AI，但必须配置使用您自己的 API 密钥。

### API Key

配置您自己的 OpenAI API Key。 [此处获取 API Key](https://platform.openai.com/account/api-keys)

- OpenAI 的 API Key 是以 sk-xxxxx 开头的一串字符。
- 如果未配置，则必须在登录后使用作者提供的 API Key 服务。
- 您还可以通过调用官方的/dashboard/billing/subscription API 来检查您输入的 API Key 是否正确。

### API 代理

- 默认情况下，使用官方 API 的地址：[https://api.openai.com](https://api.openai.com/)
- 由于某些情况，某些地区无法直接访问官方 API。如果您已经设置了自己的转发服务，可以在此处配置您的转发服务地址。我们建议使用 [Cloudflare](https://www.cloudflare.com/) 来搭建您的转发服务。

### Temperature

> 官方释义：What sampling temperature to use, between 0 and 2. Higher values like 0.8 will make the output more random, while lower values like 0.2 will make it more focused and deterministic.

在 Le-AI 中，这个设置已经被简化为只有三种模式可供选择：确定性/中性/随机性。您可以根据自己的需要调整此参数。

### Max Tokens

> 在 Le-AI 中，默认的 max tokens 为 2000。

聊天会话中单次生成的最大 Token 数量。

输入 Token 和生成 Token 的总长度由模型的上下文长度限制。
