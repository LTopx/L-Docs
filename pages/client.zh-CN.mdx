# 客户端

## Windows、Mac

![Client](/client/dark.png)

### 基本情况

Le-AI 的 Windows、Mac 客户端是采用 **Tauri + NextJS** 打包而来，目前还处于 **Beta** 版本。

- 现在以及以后都不准备提供用户系统，只能配置 API Key 使用
- 由于一些技术原因，目前没有多语言支持，只能使用英文
- 暂时没有插件系统和会话分享功能
- 因为是 Beta 版本，所以目前可能会存在一些问题，如果发现了问题，可以在 [Issues](https://github.com/LTopx/Le-AI/issues) 中进行反馈

### 下载

[下载地址](https://home.le-ai.app)

### 如何使用

因为客户端不提供用户系统，因此只能配置 API Key 来使用。

#### 使用 Le-AI Key

> Le-AI Key 是在 Le-AI Web 端生成的可以供 Le-AI 生态使用的 API Key。配置后即可获得与生成该 API Key 账号想同的权限。

- 生成：在系统右上角账户中心里的令牌页面生成。https://le-ai.app/account/token
  ![Client](/client/step-1-zh-CN.png)
- 新生成的令牌**默认是禁用状态**，需要手动启用。
- 生成完成之后，即可点击复制按钮复制 API Key。
  ![Client](/client/step-2-zh-CN.png)
- 现在把 API Key 粘贴到客户端里即可开始使用。
  ![Client](/client/step-3.png)
