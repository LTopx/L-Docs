Now you can use L-GPT more efficiently with AI characters.

### Built-in AI Characters

It includes some prompts collected by the author, as well as some excellent prompts selected from [Awesome ChatGPT Prompts](https://github.com/f/awesome-chatgpt-prompts), built into the system.

Resource reference websites:

- https://github.com/f/awesome-chatgpt-prompts
- https://www.aishort.top
- https://prompts.chat

### Stay Updated

Developers will continuously update the built-in prompts. You can check the latest prompts on the `AI Character Templates` page.

### Create Local Characters

After discovering useful or fun prompts elsewhere, you can create them locally for personal use in the future.

### Bookmarking Feature

You can bookmark your favorite prompts for easy access later.
