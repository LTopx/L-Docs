# Prompt 集市

import projectLinkImage from "../../public/change-log/v0.6.0-1.png";

<Screenshot src={projectLinkImage} alt="Project link" full />

现在您可以通过 Prompt 集市更高效的使用 L-GPT 了。

### 内置 Prompts

它包含了作者收集的一些的 prompt，以及从 awesome-chatgpt-prompts 中精选的一些优秀的 prompt，内置于系统中。

资源参考网站：

- https://www.aishort.top/
- https://prompts.chat/

### 本地创建

您在其他地方发现有用或者好玩的 prompt 后，可以在本地创建可重复使用的 prompt，并且也可以把它们分享到 Prompt 集市中。

### 用户共建

用户可以将自己的 prompt 提示到 Prompt 集市。审批通过后，其他 L-GPT 用户将能够查看并使用您提交的 prompt，帮助大家提高使用效率。

### 未来的计划

点赞，收藏，快速筛选等功能都在未来的开发计划中。
