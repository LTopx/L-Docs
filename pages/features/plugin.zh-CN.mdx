# 插件功能

## 谷歌搜索

### 简介

ChatGPT 目前的训练数据只截止到 2021 年 9 月，因此很多实时信息是无法回答的。为了解决这个问题，我们接入了谷歌搜索插件，可以让 ChatGPT 回答一些实时信息。

### 如何获取

- 点击跳转到“谷歌搜索”官网：https://programmablesearchengine.google.com
- 点击 Get started
  ![step-1.png](/features/plugin/step-1.png)
- 创建新的搜索引擎
  ![step-2.png](/features/plugin/step-2-zh-CN.png)
- 获取搜索引擎 ID
  ![step-3-1.png](/features/plugin/step-3-1-zh-CN.png)
  ![step-3-2.png](/features/plugin/step-3-2-zh-CN.png)
- 获取 API KEY
  ![step-4-1.png](/features/plugin/step-4-1-zh-CN.png)
  ![step-4-2.png](/features/plugin/step-4-2-zh-CN.png)
  ![step-4-3.png](/features/plugin/step-4-3-zh-CN.png)
  ![step-4-4.png](/features/plugin/step-4-4-zh-CN.png)

### 如何使用

将获取到的`谷歌搜索 引擎 ID` 和 `谷歌搜索 API KEY`填入到 L-GPT 插件配置当中，并“启用谷歌搜索”，即可使用。

![step-5.png](/features/plugin/step-5-zh-CN.png)
