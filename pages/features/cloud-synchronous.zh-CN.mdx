# 云同步

### 基本介绍

这是一个有争议且颇具成本的问题。以下是未来的开发方向：

- 不会提供系统设置和会话内容的实时同步功能。未来会提供支持手动同步设置和会话内容的功能。
- 在成本较低的早期阶段将提供免费同步功能。如果将来成本过高，可能会考虑收费。
