# 备份 & 云同步

### 基本介绍

允许用户手动备份和同步系统设置以及会话内容到云端，以便在不同设备上使用。

### 备份规则

- Le-AI 永远不会自动同步你的任何信息。包括但不限于会话列表、配置信息以及各种 API 密钥等。
- Le-AI 所有备份、恢复操作只能手动触发，将来也不会开发自动备份和恢复的功能。
- 所有的数据都默认存储在浏览器和设备本地，除了你自己其他任何人都无法访问这些数据。
- 当使用备份和云同步功能时，Le-AI 将采用您提供的本地加密密钥对数据在本地进行 AES 加密，然后将加密后的数据上传到云端。恢复备份时也需要相同的密钥进行解密才能正常恢复。
- 请务必妥善保管好您的密钥。如遗忘密钥则没有任何办法能够正确找回您备份的数据。

特别注意：

- 每次备份都会覆盖云端已有的备份数据，不会作合并兼容处理，之前的备份数据将会丢失。
- 每次恢复数据时都会覆盖本地的所有数据，不会作合并兼容处理。
- 目前备份只支持全量备份，即会话列表，系统配置和 API 密钥等。

### 备份额度

- 免费用户：有 50KB 的免费备份额度，超过额度后将无法备份成功。
- 高级版、团队版用户：拥有 1M 的备份额度。后续会根据用户反馈情况进行调整或开设单独的额度套餐。

### 如何操作

#### 备份

- 点击系统左下角 ☁️ 备份按钮，输入您的本地加密密钥，点击 `备份` 按钮即可。系统会通过你输入的加密密钥，采用 AES 加密算法对您的数据进行加密，然后上传到云端。

#### 恢复

- 点击系统左下角 ☁️ 备份按钮，输入您的本地加密密钥，点击 `恢复` 按钮即可。系统会通过你输入的加密密钥，将云端的加密数据取回，通过本地密钥进行解密。
