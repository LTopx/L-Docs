# Token Calculation

### Basic Definition

Using L-GPT requires consuming tokens for all text conversations, voice communications, file parsing functions, and some additional services for enhancing the user experience.

### Conversation Cost Calculation

After each conversation, the current session's token consumption will be calculated in real-time and displayed below the top title. This allows you to quickly determine the total cost of the session and whether it will exceed the [token limit]() of the language model itself.

![Conversation Cost Calculation](/basic/token-calculation/token.png)

### Complete consumption

Click on the conversation consumption area at the top of the conversation to view the total token consumption of the current conversation and some definitions and explanations related to tokens in the opened pop-up window.

![Complete consumption](/basic/token-calculation/cost.png)
