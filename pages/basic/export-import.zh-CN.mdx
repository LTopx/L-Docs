# 导入、导出

目前，Le-AI 的所有会话记录和 API 密钥都保存在本地浏览器的 localstorage 中（将来会提供云存储功能）。然而，也提供了导入和导出功能，以便用户能够快速地将本地数据转移到其他设备和浏览器中。

### 一键导出

导出 JSON 格式文件。一键导出支持导出以下内容：

- 聊天记录
- 本地配置的 API 密钥
- 各种语言模型的基本配置
- 本地自定义的 Prompt（待完善）

### 一键导入

选择您需要导入的 JSON 文件。（请注意，这将覆盖您的本地聊天历史记录和 API 密钥配置，本地自定义的 prompt 等。）
