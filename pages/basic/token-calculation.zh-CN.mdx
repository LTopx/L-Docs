# Token 消耗计算

### 基本定义

使用 L-GPT 的文本对话、语音通讯、文件解析以及一些额外的来增强用户体验的功能或服务所消耗的东西就叫做 Token。

### 会话消耗计算

每次对话后，当前会话的令牌消耗将实时计算并显示在顶部标题下方。这使您可以快速确定会话的总费用以及是否超过了语言模型本身的 [Token 限制](https://docs.ltopx.com/zh-CN/token)

![Conversation Cost Calculation](/basic/token-calculation/token.png)

### 当前会话总消耗

单击会话顶部的对话消耗区域，以查看当前会话的总代币消耗量以及在打开的弹出窗口中与 Token 相关的一些定义和解释。
