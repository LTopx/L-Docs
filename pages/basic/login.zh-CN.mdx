# 用户登录

目前支持以下几种登录方式。

![login](/change-log/v0.9.1-1-zh-CN.png)

### Github / Google

使用自己的 Github 或 Google 账户一键登录。

### 邮箱

输入邮箱账号登录。登录时，会发送一封登录邮件到您填写的邮箱钟，点击邮件中的链接即可完成登录。

![1](/basic/login/1-zh-CN.png)
![2](/basic/login/2.png)

### 账号（邮箱账号）/ 密码

- 所有的密码都采用 MD5 加密存储
- 首先需要在登录后，系统右上角进入“账户中心-修改密码”处配置初始密码
- 然后再登录界面输入邮箱账号和密码登录
