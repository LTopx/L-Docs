# Data Reset

### Reset Data

When some abnormal errors occur, you can click this button to clear the browser's local cache data and reinitialize it. Or, if you simply want to clear and reset the data, you can also achieve your goal through this function.

![Reset Data](/basic/data-reset/reset.png)

### Confirm Reset

After clicking confirm, local cache data will be cleared and the page will be refreshed.

![Confirm Reset](/basic/data-reset/confirm.png)
