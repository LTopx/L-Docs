# Token

### 什么是 Token?

Token 是 LLM 模型用于计算文本长度的基本单位。它们是字符组，有时与单词对齐，但并非总是如此。特别是当他们是标点符号或表情符号时，往往取决于它们的字符数。

### Token 如何工作?

- 根据每个语言模型的规则，根据当前会话内容的总长度计算。
- 当会话达到一定长度时，您将无法再向会话中添加任何消息。这是每个语言模型会话内容的上下文限制。
- 例如，GPT-3.5 每个会话最多允许 4,096 个令牌。GPT-4 允许每个会话在其 8K 模型中最多 8,192 个令牌，在其 32K 模型中最多 32,768 个令牌。上下文长度限制可能会在未来的各种语言模型中增加。
  如果当前会话达到上下文限制，您可以删除一些会话内容或开始一个新的会话以继续聊天。

### Token 消耗如何计算?

- 任何与聊天对话相关的内容都会产生会话费用。
- 所有消耗都是使用[openai/tiktoken](https://github.com/openai/tiktoken)计算出来的“估算值”（尽管它足够精确）
- 每个语言模型都有自己的定价，相应的官方网站有详细的定价说明。

### Token 充值

- 如果你使用自己的 API Key，那么 token 消耗则是计算到你的 API Key 中
- 如果你没有自己的 API Key，可以进行 Token 额度充值后来使用 L-GPT
  - Token 充值价格都是按照 正常价格 + 平台抽成手续费 来计算的，收费基准是至少保证作者不亏本 🤣🤣
  - 如果是高级版会员，充值 Token 会有额外赠送
